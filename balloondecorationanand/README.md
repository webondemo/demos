PROJECT NAME:
Balloons Decorations Anand - Event Decoration Website

----------------------------------------------------
ABOUT PROJECT
----------------------------------------------------
This is a modern responsive business website for
Balloon Decoration services based in Anand, Gujarat.

The website showcases:
- Event decoration services
- Portfolio gallery
- Customer reviews
- Service areas
- Booking system
- Contact options (Call, WhatsApp, Email)

It is designed to generate leads and bookings for
birthday, wedding, anniversary, baby shower and
other event decoration services.
